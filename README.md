# RAG Chatbot with Chat History (Streamlit)

This is a Streamlit app that builds a RAG chatbot with chat history using local PDF documents.

## Run locally
1. Install dependencies: `pip install -r requirements.txt`
2. Create a `.env` containing `GROQ_API_KEY=your_key_here`.
3. Put a PDF named `attention.pdf` in the project root (or update the loader path).
4. Run: `streamlit run rag_chathistory_stream.py`.
